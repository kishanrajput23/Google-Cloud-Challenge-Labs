Goto Compute Engine -> VM Instannces -> Create Instance.
   Name : <Given in the lab>

Task 2 : Configure Apache2 Web Server in your instance

SSH to 'apache' instance and run
sudo su -
apt-get update
apt-get install apache2 -y
service --status-all
